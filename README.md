# Cutting Lock

A painstaking smart lock using biometric technology and physical pain to
painlessly open the door while preventing the painful memory of valuable
things being stolen.

## Dependencies

- [Node.JS v6](https://nodejs.org/en/)

## Install

### Web

Run the commands:

```
cd web/
npm i
```

## Run

### Web

Run the commands:

```
cd web/
npm run server
```
